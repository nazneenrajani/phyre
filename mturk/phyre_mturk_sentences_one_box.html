Task Description

In this task, you will see a physics puzzle in a simulated 2D world with a constant downward gravitational force and a small amount of friction.
Each puzzle has a goal state and starts with an initial state.
A puzzle can be solved by placing one red ball so that the goal state is satisfied when the simulation runs.

You are given:
  1. The goal state expressed in one sentence.
  2. The screenshot of the initial state with the red ball already placed.
  3. The simulation demo of all collisions.
  4. The screenshots of important collisions until the goal state is satisfied.

You will answer the following questions:
  1. Describe the initial state in your own words. You need to describe the object details (shape, color, position).
  2. Describe the whole process in your own words. You need to describe the collision details (which two objects collide, in which direction, to have what effect).
Note:
  1. Check the task example below before you begin.
  2. There are multiple screenshots at the same last time step if the number of screenshots is not enough.
