Task Description

In this task, you will see a physics puzzle in a simulated 2D world with a constant downward gravitational force and a small amount of friction.
Each puzzle has a goal state and starts with an initial state.
A puzzle can be solved by placing one red ball so that the goal state is satisfied when the simulation runs.

You are given:
  1. The goal state expressed in one sentence.
  2. The screenshot of the initial state with the red ball already placed.
  3. The screenshots of all collisions during the simulation until the goal state is satisfied.

You will answer the following questions:
  1. Select collisions that are important to reach the goal state by clicking the Yes/No button under each screenshot.
Note:
  1. Check the task example below before you begin.
  2. Please select all and only the collisions that are causally related to the placement of the red ball and are necessary for the completion of the goal.
  3. Only one selection per important collision. If there are multiple consecutive screenshots for the same collision that you think is important, only choose the first of them.
  4. The last important collision to choose should be the goal state.
